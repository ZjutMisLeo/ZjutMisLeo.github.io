  1. Java final finally finalize

    Final is a keyword. Final is used to apply restrictions on class, method and variable. Final class can’t be inherited, final method can’t be overridden and final variable value can’t be changed.

    Finally is a block. Final is used to apply restrictions on class, method and variable. Final class can’t be inherited, final method can’t be overridden and final variable value can’t be changed. (try catch)

    Finalize is a method. Finalize is used to perform clean up processing just before object is garbage collected.

Intersection of Two Linked Lists

区别于LC上的原题，这里会问链表有环的情况

  1. 两表都无环，直接用LC解法。

  2. 一表有环一表无环，不会相交。(都相交了，怎么可能A有环B没有)

  3. 两表都有环。这时候需要先分别找到两个链表的入环节点。

    • 如果两节点相同，则进入case1，说明相交点在环之前，两表相交
    • 如果两节点不同，可能是相交点在环上，需要从一个节点开始遍历一遍环，看环上是否存在另一个链表的入环点
